How Remote Cache Works:

  • Detects clients with 'loc' capability (like LyrPlay iOS app)
  • Converts local file paths to HTTP streaming URLs
  • Wraps HTTP URLs in file:// format for remote caching
  • Allows mobile clients to download and cache music files
  • Falls back to normal streaming for clients without 'loc' capability

Compatible Apps: LyrPlay for iOS, custom Squeezelite builds with remote caching
